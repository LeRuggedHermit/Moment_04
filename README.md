# Moment_04

Innehåller en webbtjänst som använder en modell, Song, för att hantera data in och ut ur en databas. Till hjälp för detta finns en datakontext - i datamappen SongContext. 
En controller har skapats för att hantera CRUD-funktionalitet (Create Read Update Delete). 

Som databas används här SQLite.

använder i övrigt följande NuGet paket:
Microsoft.VisualStudio.Web.CodeGeneration.Design
Microsoft.EntityFrameworkCore.Design
Microsoft.EntityFrameworkCore.SQLite
Microsoft.EntityFrameworkCore.SqlServer
